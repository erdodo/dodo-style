# Config (Yakında)

## Kullanım
Projenizin ana dizininde `dodo.config.js` dosyası oluşturun.

## Varsayılan Component Ayarları
```js
module.exports={
    default:{
        button:{
            type:"danger",
            plain:true
        }
    }
}
```

## Özel Component Oluşturma
```js
module.exports={
    components: {
        button: {
            red: {
                type: "ssdafdsdf",
                plain: "asdkfmdk",
                outline: "asfglmdfsmdfşl"
            }
        }
    }
}
```
