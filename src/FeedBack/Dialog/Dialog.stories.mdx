# Dialog




## Konum Örneği
<Example1 location={"center"} />

```jsx
<Button type={"primary"} onClick={()=>{ setVisible(true) }}>Dialog aç</Button>
<Dialog open={visible} location={location} onClose={()=>{ setVisible(false) }} title={"Dialog Başlığı"} >

        Merhaba
    </div>
</Dialog>
```
