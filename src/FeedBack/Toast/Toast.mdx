# Toast
## Kullanım
ToastContainer parent component içerisinde tek sefer tanımlamanız yeterlidir.Daha sonra toast fonksiyonu ile kullanabilirsiniz.
```jsx
import { ToastContainer, toast } from 'dodo-style'

export default function Foo(){
    return(
        <>
            <ToastContainer/>
        </>
    )
}
```

## Config
- Global ayarlama
```jsx
    <ToastContainer
        position="top-right"
        autoClose={5000}
        hideProgressBar={false}
        newestOnTop={false}
        closeOnClick
        rtl={false}
        pauseOnFocusLoss
        draggable
        pauseOnHover
        theme="dark"
    />
```
- Local ayarlama
```js
    toast.info('Bu kadar basit', {
        position: "top-right",
        autoClose: 5000,
        hideProgressBar: false,
        closeOnClick: true,
        pauseOnHover: true,
        draggable: true,
        progress: undefined,
        theme: "dark"
    });
```
